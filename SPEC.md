# 子供のおこづかい管理アプリ【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Allowance Manager
**バージョン**: 2.0.0
**更新日**: 2026-06-03
**URL**: https://appadaycreator.com/allowance-manager/

お小遣いの収支を記録してお金の使い方を学習。グラフで可視化。登録不要・完全無料でご利用いただけます。

## 主な機能

### v2.0.0 新機能（2026-06-03追加）
- **グラフ表示**: カテゴリ別支出割合（円グラフ）と月別推移（折れ線グラフ）
- **日付選択機能**: 過去の日付でも記録可能（当日以外のデータも対応）
- **月別フィルタリング**: 特定の月のデータのみ表示・分析
- **記録編集機能**: 誤入力時に削除せず編集で修正可能

### 基本機能
- 収入・支出の記録（ワンタップ切り替え）
- 金額・メモ・カテゴリ選択による詳細記録
- リアルタイムサマリー（合計収入、支出、残高の可視化）
- 記録一覧表示（日付、カテゴリ、内容、金額を表示）
- CSVダウンロード機能
- 全データ削除機能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 子供のおこづかい管理アプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
